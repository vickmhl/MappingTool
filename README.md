# 竞对组织架构 Mapping 工具

一个纯前端、本地处理的 HR mapping 工作台。它可以导入电话转写文本和 PPTX，自动抽取组织、岗位、人员、汇报关系和变更候选，经过确认后生成可筛选组织图，并导出 PPTX、PNG 或加密项目包。

## 设计原则

- 敏感数据不上传：资料解析、抽取、存储、导出都在浏览器内完成。
- 不依赖外部大模型：V1 使用规则、词典、证据片段和人工确认队列。
- 低成本发布：可直接部署到 GitHub Pages，也可把 `dist` 放到公司内网静态站点。
- 文件协作：项目可导出为密码保护的 `.mapping.zip`，再导入到另一台电脑。

## 本地运行

```bash
npm install
npm run dev
```

## 构建与测试

```bash
npm run test
npm run build
```

## 支持的输入

- `.txt` / `.md`：电话转写、访谈纪要、公开资料摘录。
- `.pptx`：解析可选中文字；图片型页面会尝试浏览器本地 OCR，失败时保留为待人工补录提示。

## 组织图画布与样例

- 组织图拆分为“正式组织架构图”和“探索画布”。正式图按层级、BU 泳道、直角汇报线自动排版；探索画布支持手工拖动和保存位置。
- 支持高管版、招聘版、详细版三套画布布局，可分别适配领导汇报、招聘 mapping 和组织诊断。
- 标准节点卡会展示姓名、职位、部门、层级、证据置信度、管理跨度、收起下属数量、近期变动和信息新鲜度。
- 负责人焦点会自动展示上级链、同级和下属；近 90 天变化视图会突出新增、调岗和其他变动人员。
- 导出中心支持 PPT 16:9、A4 横版和长图三种组织图导出版式，并提供发布前检查。
- 内置小规模样例和“大规模地图业务虚拟样例”。后者包含 5000+ 虚拟人员、5000+ 汇报线，用于测试大组织筛选、层级展示和画布性能。
- 大规模样例不对应任何真实公司的组织架构，只用于功能演示与压力测试。

## GitHub Pages 发布

项目已包含 `.github/workflows/pages.yml`。推送到 GitHub 仓库的 `main` 分支后，Actions 会执行测试、构建并发布 `dist` 到 GitHub Pages。

```bash
npm ci
npm run test
npm run build
```

如果公司内网不方便访问 GitHub Pages，也可以把 `dist` 目录直接交给内网静态站点托管。

## HR 工作台能力

- 候选确认支持按来源筛选、接受高置信任职、忽略弱证据，并提示同一人的任职冲突。
- 候选冲突助手支持保留最高置信候选并忽略其余重复任职。
- 人员组织页支持重点人才池筛选、手动加入/移出重点人才池、人员详情、直属下级展开、任职记录和证据时间线。
- 证据时间线支持按强证据、待验证、弱证据过滤。
- 导出中心支持 PPT/PNG 预览、当前画布视图提示、内部版/领导版/外发版脱敏预设，以及姓名、公司/组织、来源、备注/说明的细粒度脱敏。
- 管理简报页提供汇报准备度、覆盖度、可信度、时效性、确认度、风险雷达、证据健康度和咨询式下一步动作。
- 管理简报新增“人才机会与业务解释”，把管理跨度、重点人才池、组织孤点、近期变更等信号翻译成业务含义和推荐动作。
- 管理简报新增战略业务线地图、关键岗位缺口地图、公司/BU 对比、高招作战清单、人才流动热力图、组织变化预警、导入质量诊断、假设与口径、汇报故事线。
- 准备度评分权重可按管理汇报、高招 Mapping、组织诊断等场景调整，也可以记录当前权重口径。
- 来源可信度可按文本、Markdown、PPTX、OCR、项目包调整；准备度中的可信度会按来源权重计算。
- 汇报模板支持高层汇报、招聘 Mapping、组织诊断、外发脱敏四种口径，一键同步画布视图、层级、节点上限、置信度和脱敏设置。
- 导出中心支持内部协作、受限汇报、外发脱敏三档敏感分级，自动同步姓名、公司、来源和备注脱敏策略。
- PPTX 导出自动包含管理结论、组织风险、证据健康度、关键人员清单、近期变更与组织图页面。
- 本地操作审计会记录导入、候选确认、人才池标记、画布保存、模板切换、脱敏设置和导出动作，方便复盘口径变化。
- 仪表盘提供“3步极简引导”和“下一步待办”，按照导入、确认、复核、标记人才池、调整组织图的顺序提示初级 HR 操作。
- 组织图支持负责人焦点自动匹配、汇报放大模式，并可恢复原层级/节点上限。
- 人员详情支持直属下级展开与搜索，人员清单固定表头，重点人才池筛选状态更明显。
