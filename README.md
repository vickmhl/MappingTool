# 竞对组织架构 Mapping 工具

纯前端、本地处理的 HR 竞对组织架构 mapping 工具。资料只在浏览器本地解析、确认、存储和导出，不上传服务器。

## 核心流程

1. 上传 `.txt`、`.md`、`.pptx`、图片资料。
2. 确认识别出的人员、组织、任职、汇报线和变更。
3. 在组织图中切换汇报模式或招聘模式。
4. 导出 PPTX、PNG 或加密 `.mapping.zip` 项目包。

## 组织图模式

- 汇报模式：只保留头部到下 3 级，节点展示部门、一号位、人数和备注。
- 招聘模式：尽可能展开人员结构，支持搜索、负责人聚焦、展开更多和折叠部门。
- 图形样式：常规架构图、树状图。

## 本地运行

```bash
npm install
npm run dev
```

## 构建与测试

```bash
npm run test
npm run build
```

## 发布

仓库包含 GitHub Pages workflow。推送到 `main` 后会执行测试、构建并发布 `dist`。

请不要把真实竞对资料提交到仓库。仓库只放代码和虚拟样例。
